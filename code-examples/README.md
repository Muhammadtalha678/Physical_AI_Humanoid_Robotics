# Code Examples for Physical AI & Humanoid Robotics Textbook

This directory contains code examples referenced in the Physical AI & Humanoid Robotics textbook. The examples are organized by topic area:

## Directory Structure

- `ros2-examples/` - ROS 2 programming examples
- `isaac-examples/` - NVIDIA Isaac Sim and AI robotics examples
- `capstone/` - Capstone project examples and solutions

## ROS 2 Examples

The ROS 2 examples demonstrate fundamental concepts covered in Weeks 3-5:

- `simple_publisher.py` - Basic publisher node example
- `simple_subscriber.py` - Basic subscriber node example
- More examples will be added as the curriculum develops

## Isaac Sim Examples

The Isaac Sim examples demonstrate AI-powered robotics concepts from Weeks 8-10:

- `simple_isaac_example.py` - Basic Isaac Sim integration example
- `rl_cartpole_example.py` - Reinforcement learning example adapted for Isaac Sim concepts
- More examples will be added as the curriculum develops

## Running Examples

To run these examples, you'll need to have the appropriate ROS 2 and Isaac Sim environments set up. See the respective sections in the textbook for setup instructions.
